# DC-TEP-with-New-Bus

This repository contains code, datasets, and computational results for manuscripts titled "Big-M Tightening and Cutting-Plane Methods for DC Transmission Expansion with New-Bus Subnetworks."

## Contents

- `code/`: Python implementation (DC_TEP_Solver.py)
- `data/`: input datasets
- `results/`: computational results in Excel format

## Test case source

The computational experiments are based on the **Texas Synthetic Grid – 6717 Buses** test case (with a few modifications), provided through the **Electric Grid Test Case Repository** maintained by the **Texas A&M Engineering Experiment Station (TEES)**:
https://electricgrids.engr.tamu.edu/electric-grid-test-cases/

35 instances are generated by randomly perturbing each positive bus demand, scaling it by a factor drawn from a uniform distribution between -15% and 25%, and setting any negative resulting value to zero.

Five solution methods are tested for the DC-TEP problem with new-bus subnetworks. These include a basic approach with one common big-M value for all candidate lines (NVI), two stronger bounding approaches that use shortest-path information and then LSPC when needed (SP and LSPC, respectively), and two cut-and-branch methods built on the SPP+LSPC bounding scheme. Among the cut-and-branch methods, one uses random cut selection as a baseline (RandomVI), while the other uses the proposed algorithm to identify and add promising PVIs (VI).
